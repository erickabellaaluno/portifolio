Objetivo
==============
Explorar o uso de LLMs para apoiar a formulação de problemas computacionais, ao mesmo tempo em que se desenvolve o raciocínio lógico e a capacidade de resolução algorítmica de forma independente.

Descrição da Atividade
==============
Cada aluno deverá propor dois problemas envolvendo vetores/listas, com o apoio de uma LLM (por exemplo, ChatGPT). Os problemas devem estar claramente relacionados à área do seu curso de graduação, como:
Engenharia Mecânica
Engenharia Elétrica
Engenharia de Telecomunicações
Os problemas devem refletir situações realistas ou plausíveis dentro dessas áreas.

Requisitos
==============

Formulação dos Problemas
Utilizar uma LLM para ajudar a definir os dois problemas.
Os problemas devem envolver operações com vetores/listas (ex: busca, soma, média, filtragem, comparação, etc.).
Devem estar contextualizados na área do curso do aluno.
Resolução dos Problemas
Resolver os dois problemas manualmente, utilizando:
papel e caneta
linguagem própria (descrição algorítmica, passo a passo)
Não é permitido usar código gerado por LLMs ou ferramentas de programação nesta etapa.


Entrega
==============
Uma imagem (foto ou scan) contendo:
os dois problemas definidos
as respectivas resoluções manuscritas



Critérios de Avaliação
============================
Clareza e relevância dos problemas propostos
Adequação ao uso de vetores/listas
Correção e coerência das soluções
Qualidade da explicação algorítmica (organização, lógica, clareza)


Observações Importantes
============================
O uso de LLMs é permitido apenas na etapa de criação dos problemas.
A resolução deve refletir entendimento próprio, sem dependência de ferramentas automáticas.
Problemas genéricos ou desconectados da área do curso poderão ter pontuação reduzida.
