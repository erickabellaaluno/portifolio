Objective
=========
The objective of this activity is to use Artificial Intelligence and Python programming to create useful artifacts for the portal, contributing to the expansion, update, and enrichment of the content available about the city of Alegrete-RS.

Each student must use one or more LLMs (ChatGPT, Claude, Gemini, Copilot, Grok, DeepSeek, among others) to assist in developing a Python solution capable of performing one of the tasks described below.

Tasks
==============
Each student must choose one of the options below.

(a) Event schedule update
-----------------------------------------------
Develop a Python script that fetches the schedule available at:
https://eventos-alegrete.glide.page/dl/programacao
and identifies events, activities, or information that are not yet registered in the events/programming section of the portal alegrete.org.
The result should present a report (or a simple TXT file) containing the items found and suggested updates for the portal.

(b) Automated image search
-----------------------------------------------
Develop a Python script capable of finding high-quality images related to Alegrete-RS for use in:
Home page cards;
Themed pages;
News;
Events;
Tourism;
Culture;
Sports;
Education.
The result should include an organized gallery containing links, sources, and metadata for the images found.

(c) Photographic gallery of Alegrete
-----------------------------------------------
Develop a solution that automatically creates a photo gallery of the city of Alegrete-RS.
The gallery may be built from:
Public sources;
Institutional collections;
Free image banks;
Social networks (respecting licenses and copyrights).
The result should include captions, location (when available) and links to the original sources.

(d) Automatic promotional video generation
-----------------------------------------------
Using Python and AI tools, develop a solution capable of processing the video currently used on the portal's home page and automatically generating a short promotional video of 10 to 30 seconds.
The video should highlight positive aspects of the city and the portal, and may include:
Transitions;
Soundtrack;
Captions;
Visual effects;
AI-generated narration.

(e) Sports events catalog
-----------------------------------------------
Develop a catalog of sports events held in Alegrete-RS.
The catalog should contain:
Event name;
Sport modality;
Date;
Location;
Photos or images;
Link to the official event page;
Associated social networks (Instagram, Facebook, etc.);
History of previous editions (when available).
The result should be organized in a structured format (JSON, CSV, database or web page).

(f) Free theme
-----------------------------------------------
Propose and develop any other solution that can contribute to the portal alegrete.org.
The proposal must be previously described and approved by the instructor.

Mandatory use of AI

Development must be carried out with the support of one or more LLMs.

The student must:
Define the problem;
Prepare prompts;
Iterate over the AI responses;
Correct errors;
Test the solution;
Refine the code produced.

Solutions produced in a single AI interaction without validation or human intervention will not be accepted.

Deliverables
-----------------------------------------------
(i) Source code
GitHub repository containing:
Python code;
Dependencies;
README.md;
Execution instructions;
Usage examples.

(ii) Short document (TXT, PDF or Markdown) of up to 1 page containing:
Objective of the solution;
LLM(s) used;
Main difficulties encountered;
Results obtained;
Possible future improvements.

Final Objective of the Task
======================
Produce catalogs, databases, images, videos, galleries, content and tools that can be effectively used to enrich the portal alegrete.org and make it an increasingly complete source of information about the city of Alegrete-RS.
