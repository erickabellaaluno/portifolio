Objetivo
=========
O objetivo desta atividade é utilizar Inteligência Artificial e programação Python para criar artefatos úteis ao portal , contribuindo para a ampliação, atualização e enriquecimento do conteúdo disponível sobre a cidade de Alegrete-RS.

Cada aluno deverá utilizar uma ou mais LLMs (ChatGPT, Claude, Gemini, Copilot, Grok, DeepSeek, entre outras) para auxiliar no desenvolvimento de uma solução em Python capaz de realizar uma das tarefas descritas a seguir.

Tarefas
==============
Cada aluno deverá escolher das opções abaixo.

(a) Atualização da programação de eventos
-----------------------------------------------
Desenvolver um script Python que consulte a programação disponível em:
https://eventos-alegrete.glide.page/dl/programacao
e identifique eventos, atividades ou informações que ainda não estejam cadastrados na seção de eventos/programação do portal alegrete.org.
O resultado deve apresentar um relatório (ou arquivo TXT simples) contendo os itens encontrados e possíveis atualizações sugeridas para o portal.

(b) Busca automatizada de imagens
-----------------------------------------------
Desenvolver um script Python capaz de localizar imagens de alta qualidade relacionadas a Alegrete-RS para utilização em:
Cards da página inicial;
Páginas temáticas;
Notícias;
Eventos;
Turismo;
Cultura;
Esporte;
Educação.
O resultado deve incluir uma galeria organizada contendo links, fontes e metadados das imagens encontradas.

(c) Galeria fotográfica de Alegrete
-----------------------------------------------
Desenvolver uma solução que crie automaticamente uma galeria de fotos da cidade de Alegrete-RS.
A galeria pode ser construída a partir de:
Fontes públicas;
Acervos institucionais;
Bancos de imagens livres;
Redes sociais (respeitando licenças e direitos autorais).
O resultado deve incluir legendas, localização (quando disponível) e links para as fontes originais.

(d) Geração automática de vídeo promocional
-----------------------------------------------
Utilizando Python e ferramentas de IA, desenvolver uma solução capaz de processar o vídeo atualmente utilizado na página inicial do portal e gerar automaticamente um vídeo promocional curto, com duração entre 10 e 30 segundos.
O vídeo deve destacar aspectos positivos da cidade e do portal, podendo incluir:
Transições;
Trilha sonora;
Legendas;
Efeitos visuais;
Narração gerada por IA.
(e) Catálogo de eventos esportivos
-----------------------------------------------
Desenvolver um catálogo de eventos esportivos realizados em Alegrete-RS.
O catálogo deve conter:
Nome do evento;
Modalidade esportiva;
Data;
Local;
Fotos ou imagens;
Link para a página oficial do evento;
Redes sociais associadas (Instagram, Facebook, etc.);
Histórico de edições anteriores (quando disponível).
O resultado deve ser organizado em formato estruturado (JSON, CSV, banco de dados ou página web).

(f) Tema livre
-----------------------------------------------
Propor e desenvolver qualquer outra solução que possa contribuir para o portal alegrete.org.
A proposta deve ser previamente descrita e aprovada pelo professor.Uso obrigatório de IA

O desenvolvimento deverá ser realizado com apoio de uma ou mais LLMs.

O aluno deverá:
Definir o problema;
Elaborar prompts;
Iterar sobre as respostas da IA;
Corrigir erros;
Testar a solução;
Refinar o código produzido.
Não serão aceitas soluções geradas em uma única interação sem validação ou intervenção humana.Entregáveisa) Código-fonte
Repositório GitHub contendo:
Código Python;
Dependências;
README.md;
Instruções de execução;
Exemplos de uso.
ENTREGAR
-----------------------------------------------
(i) Documento curto (TXT, PDF ou Markdown), de no máximo 1 página, contendo:
Objetivo da solução;
LLM(s) utilizada(s);
Principais dificuldades encontradas;
Resultados obtidos;
Possíveis evoluções futuras.
(ii) Código Python utilizado para desenvolver a tarefa.

Objetivo Final da Tarefa
======================
Produzir catálogos, bases de dados, imagens, vídeos, galerias, conteúdos e ferramentas que possam ser efetivamente aproveitados para enriquecer o portal alegrete.org e torná-lo uma fonte cada vez mais completa de informações sobre a cidade de Alegrete-RS.
