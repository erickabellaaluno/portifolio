Cada aluno deverá escolher, resolver e entregar em aula um dos dois problemas abaixo:

Problema 01: Análise de Tensões em Treliças Planas
https://docs.google.com/document/d/1ZzcQWYkWz8CinTijmyT3aSI_mhQEC1rkV7eaXoxg9Aw/edit?usp=sharing

Problema 02: Análise de Estabilidade e Custo em Sistemas de Abastecimento de Água
https://docs.google.com/document/d/1o63xlRkI2A10GSfN0XG_OgASFRehFWQcxsr8oPV4ABg/edit?usp=sharing

A solução deve ser implementada em Python, utilizando vetores/listas de tamanho variável, definidos a cada execução. Os resultados devem ser apresentados em forma de tabela e também por meio de gráficos visuais.

Além da implementação, cada aluno deverá comparar a resolução do mesmo problema utilizando o mesmo prompt em pelo menos três LLMs distintas, como ChatGPT, Claude, Gemini, Copilot ou DeepSeek. A comparação deve considerar se o código executa corretamente, se os cálculos estão adequados, se os gráficos são coerentes, se há uso correto de listas/vetores e se a solução é clara e bem organizada.
